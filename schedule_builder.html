How It Works

***Please note that this is simply a proof of concept site to showcase a new schedule building idea. Many aspects does not work properly or at all such as the login screen.***


This demo simulates a schedule building concept designed for univeristies, specifically the University of Central Florida.

It works by simulating a list of classes with data such as a name, code, time, days, etc., and automatically placing them in the top bar if they are in person, and the bottom bar if they are online. The user can select which classes they want in the schedule, and the program will automatically build a schedule based on the selected classes.

This program is good for debating which of the same class but different section should be chosen based on other classes
